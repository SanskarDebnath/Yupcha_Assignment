# 🚀 From Sanskar Debnath @ FAST API MANAGEMENT FOR Yupcha Softwares Pvt. Ltd

## 📌 Features
✅ Create a user (POST)
✅ Retrieve all users (GET)
✅ Retrieve a specific user (GET by ID)
✅ Update a user (PUT)
✅ Delete a user (DELETE)

## 🛠 Tech Stack
- **FastAPI**
- **Pydantic**
- **Postman (for testing)**
---

3️⃣ **Install dependencies**
```sh
pip install fastapi uvicorn
```

4️⃣ **Run the FastAPI server**
```sh
uvicorn main:app --reload
```
The API will be available at: **`http://127.0.0.1:8000`**
